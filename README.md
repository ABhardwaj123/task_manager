# Full End-to-End Task Manager

A backend-focused task manager application built to learn and implement core backend development concepts using FastAPI and PostgreSQL.

This project is being developed as a hands-on learning project covering:

* REST API development
* Routing and API endpoints
* Authentication and Authorization
* JWT-based authentication
* PostgreSQL database integration
* SQLAlchemy ORM
* CRUD operations
* Database relationships
* Backend project structuring
* Environment variable management
* Request validation and schema handling

## Tech Stack

* FastAPI
* PostgreSQL
* SQLAlchemy
* Uvicorn
* Python

## Goal of the Project

The main goal of this project is to gain practical backend development experience by building a complete API-driven application from scratch instead of only learning concepts theoretically.
